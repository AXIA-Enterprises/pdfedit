# Code of Conduct

PDFEdit follows the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## In short

Be respectful. Assume good faith. Disagree about the work, not the person. Help newer contributors. Step back when a thread gets heated.

## Scope

This code applies to all project spaces — issues, pull requests, discussions, and any other place where someone is participating in PDFEdit on behalf of the project.

## Reporting

If you experience or witness behavior that does not meet this standard, please report it privately:

- Open a private security advisory at <https://github.com/AXIA-Enterprises/pdfedit/security/advisories/new> (the Security tab also accepts non-vulnerability private reports), **or**
- Open a regular issue and tag `@AXIA-Enterprises` if you prefer a public-but-attributed report.

Reports will be reviewed promptly and handled with discretion. Maintainers who violate this code may be removed from the project.
